# NutriVisionary

## Health App Project

### Overview
NutriVisionary is a health application built using Java and Gradle. This application aims to help users manage their nutritional intake, track their health metrics, and make informed dietary choices. 

### Features
- User authentication
- Track daily food intake
- Generate meal plans based on user preferences
- Health metrics dashboard

### Installation
To run NutriVisionary, follow these steps:
1. Clone the repository: `git clone https://github.com/ChishtiShaheer/NutriVisionary.git`
2. Navigate to the project directory: `cd NutriVisionary`
3. Build the project using Gradle: `./gradlew build`
4. Run the application: `./gradlew run`

### Technologies Used
- Java
- Gradle
